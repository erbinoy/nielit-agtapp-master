4.4.1 Using Basic Formulas

In Microsoft Excel, cell can have normal data or a formula entry. Formulas help us to analyze data on a worksheet. With a formula we can perform operations, such as addition, multiplication and comparison on worksheet values. We can use a formula when we want to enter calculated values on a worksheet.

A Microsoft Excel formula always begins with an equal sign (=). A formula is an equation whose result is shown in the cell. A formula can contain one or more Cell Addresses, Functions, Constants and Operators. There are seven basic mathematical operators which can be used for calculations.

Operator      Operation

             *      Multiplication

             +      Addition

             -      Subtraction

             /      Division

             \      Integer Division

             ^      Exponentiation

           Mod   Modulus
