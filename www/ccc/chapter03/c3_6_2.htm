 3.6 Table Manipulation
3.6.2 Changing cell width and height


You can change the width and height of a cell in a table by performing the following steps :

1. Click the cell that you want to change the height/width.

2. Click Layout tab, present at Table tools.

3. In the Cell Size tab, set the points of height/width as you desired.

OR

You can use the AutoFit option to fit the table cell according to its contents.

Three options are available in AutoFit option they are :

AutoFit Contents, AutoFit Window and Fixed column width.
