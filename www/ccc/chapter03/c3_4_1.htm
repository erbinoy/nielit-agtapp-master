 3. ELEMENTS OF WORD PROCESSING
 3.4 Text Creation and manipulation
 3.4.1 Document Creation

When you start Word 2013 and click Blank document on the Start screen, a new document opens in the program window, ready for you to enter your content. You can also create a new document while Word 2013 is running. each new document displays a default name (such as Document1, Document2, and so on) on the Title bar until you save it with a more meaningful name. The cursor, a blinking vertical line in the upper-left corner of the page, shows where the next character you type will appear. When the cursor reaches the right margin, the word you are typing automatically moves to thenext line. Pressing the Enter key starts a new paragraph.

To create a new document:

1. Click the File tab, and then click New. The New page of the Backstage view opens. displaying thumbnails of the available templates.

2. In the right pane, click Blank document. A new, blank document opens in a new window.

Or you can create a new document by pressing Ctrl+N.
