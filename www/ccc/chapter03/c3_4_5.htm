 3.4 Text Creation and manipulation
 3.4.5 Font and Size selection

A font defines the overall appearance or style of text learning. Font size controls the height of the font. The default font in new Word 2013 documents is Clibri; the default size is 11 points.

To change the font :

1. Select the text that you want to format.

2. On the Home tab, in the font group, click the font arrow and select the desaired font from the list.

To change the font size:

1. Select the text that you want to format.

2. On the Home tab, in the font group, click the font size arrow and select the desired font size from thr list. If a font size you want is not listed in the Font size list, Click in the font size box, type the desired number, and then press the Enter key.
