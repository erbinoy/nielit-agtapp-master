 3.5 Formatting the Text
3.5.3 Changing case

To change case of selected text or words in a document, do the following steps :

1. Select the text for which you want to change the case.

2. On the Home tab, in the Font group, click Change Case.

3. Choose an option from the dropdown list, which includes Sentence case, lower case, UPPER CASE, Capitalize Each Word, and tOOGLE cASE.

NOTE : To change case by using a keyboard shortcut, press SHIFT+F3 until the style you want-title case, all caps, or lower case -is selected.
