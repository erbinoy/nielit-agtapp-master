3.6.5 Border and shading

Bordering of a table :

You can set apart text from the rest of the document by adding border and shading. You can add border to any side of the text or all sides to make a box.

To add border:

1. Select the text or cell to which you want to add borders.

2. On the Design tab, in the borders group,click the Borders button to apply the most recently used border, or click the Borders arrow and select a defferent border from the menu.

NOTE: You can remove all borders from selected text by clicking the Borders arrow,and then clicking No Border on the menu.

Shading of a table:

You can set shades to a specific cell or text of a table by performing the following steps :

1. Select the text or cell of the table to which you want to apply shading.

2. On the Design tab, in the Table Styles group, select the color from Shading option.
