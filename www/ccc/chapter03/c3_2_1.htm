 3.2 Word Processing Basics
3.2.1 Openning Word Processing Package 2013

You can open Microsoft Word Processing Package 2013 by clicking Microsoft office word document shortcut from popup menu under All apps menu present at Start menu.

The steps are:

Start > All Apps > Microsoft Office > Microsoft Office Word 2013

In the right pane, click Blank Document. A new, blank document opens in the program window.
