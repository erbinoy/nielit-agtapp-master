 6.3 Configuring Web Browser
Configuring Web Browser

You can change the look and layout of web browser. To configure web browser follow the below listed steps :-

1. Click the Tools menu and click Internet Options.The Internet Options Dialog Box appears.

2. Homepage is the first webpage that will be dispalyed when you open your browser.To set a Home Page type the address of the page which you want to make homepage in the HomePage Address TextBox.

3. History folder contains links to recently viewed web pages. you can specify the number of days to keep pages in history. You can also delete history by clicking on "Clear History" button.

4. To change Text and background color , click on the "Colors" button. The "Colors Dialog Box" appears where you can select the desired color for text.

5. To change the font of text on the webpages, click on the "Fonts" Button. The "Font Dialog Box" appears where you can select the desired font for the text.
