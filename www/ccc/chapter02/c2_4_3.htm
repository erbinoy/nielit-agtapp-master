 2.4 Operating System Simple Setting
 2.4.3 To Add or Remove a Windows Component
Steps to Add a Windows Component (Application Software) :

1. Double click on the icon of the Application that you want to install or add.

2. An Administrator permission will be required and a Dialog box will appear.

3. Click Yes.

4. Follow the instructions until the installation is completed.

Steps to Remove a Windows Component (Application Software) :

1. Open Control panel.2. Double click on Programs and Features.

3. Right click on the Icon or name of the Application software.

4. Click Uninstall, and follow the instructions until the uninstallation is completed.
