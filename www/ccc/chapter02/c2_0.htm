2.0 Introduction

In the previous chapter, you have learnt about the characteristics of a Computer System and its components. You have also learnt about System Software and Application Software.

In the current chapter, you will learn about a very important System Software viz. Operating System. We will use Windows to explain various aspects of an operating system. Microsoft has produced a number of operating systems. Windows 95/98/ME/XP/Vista, Windows NT, Windows 2000/2003, Windows 8, Windows 10 are some of them.
