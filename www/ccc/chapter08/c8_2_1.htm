8.2 Basics
8.2.1 Using Power Point

To start making a presentation, you have to open the Powerpoint program. There are two methods for opening it.

Method 1

1. Click on Start button.

2. Click All Apps.

3. Open Microsoft Office.

4. Open Microsoft Powerpoint 2013.

Method 2

Click at Powerpoint shortcut,if available at desktop.

On starting powerpoint, the screen that appears prompts for a new presentation creation using Auto Content wizard, Template or a Blank presentation.

Now you can choose an autolayout..

The presentation is opened in Normal View and it shows mainly two panes : one showing Outline View and another showing its Slide View.

You can open an existing presentation also if you have already made any.
