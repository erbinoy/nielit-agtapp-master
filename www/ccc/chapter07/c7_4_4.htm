 7.4 Advance email features
7.4.4 Sending Soft Copy as attachment

Using Windows Mail or Outlook Express, it is easy to send text messages to friends, colleagues and strangers. You can even use fancy stationery or insert pictures in your messages. You can also add any file to your emails and send it as an attachment to anybody equipped with an email address.

To send a file attached to an email with Windows Mail or Outlook Express:

1. Create a new message.

2. Click the Attach button.

3. You will recognize usual file-locating window of your computer. Find the file you would like to attach.

4. You can highlight and attach multiple files at once by holding down the Ctrl key while you click on them with the mouse.

5. Click Open or Attach, whichever you see.

Sending Multiple Files with Ease and Elegance :

If you want to send more than one file with an email, you can also pack them neatly into one ZIP archive. And if you have the folder containing the file you want to attach already opened in Windows Explorer, you can alternatively, but easily, attach it by drag-and-drop.
